TYPES OF ARRAY METHODS

  • POP():IT IS USED TO REMOVE ARRAY ELEMENTS FROM LAST
  • PUSH():IT IS USED TO ADD ARRAY ELEMENTS IN THE LAST
  • SHIFT METHOED():IT REMOVES ARRAY ELEMENTS FROM STARTING
  • SHIFT UP METHOD():IT ADDS ARRAY ELEMENTS FROM STARTING
  • JOIN() & TO STRING():IT IS USED TO CONVERT ARRAY TO STRING
  • FILTER():IT IS ONE OF THE HIGHER ORDER FUNCTION WHICH IS USED TO FILTER THE ARRAY ELEMENTS BASED ON CONDITION.
  • MAP():IT IS A FUNCTION USED FOR CALLING HIGHER ORDER FUNCTION FOR OPERATIONS(+,-,*,/,%, etc..........)
  • SLICE():BY USING , WE CAN CREATE NEW ARRAY FROM EXISTING ARRAY
  • SPLICE():IT REMOVES/SELECTS CERTAIN AMOUNT OF ELEMENTS
  • FLAT():IT CONCATENATES SUB ARRAY
  • INDEX IN && INDEX OF
  • KEYS() && VALUES() && ENTRIES()
  • SOME()
  • EVERY()
  • FIND()
  • FIND INDEX()
