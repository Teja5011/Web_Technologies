1. NAMED FUNCTION

    1. SYNTAX: FUNCTION FunctionName(){
      statments;
      }
      FunctionName(); //call back Function

    2. function with arguments (or) function with parameters
      SYNTAX:function functionName(para1,para2,.......,paraN){
      statments;
      }
      functionName(arg1,arg2,......,argN);

    3. function with return type
      SYNTAX:function functionName(para1,para2,.......,paraN){
      statments;
      return type
      }
      functionName(arg1,arg2,......,argN);
      statments;
  2. ANONOMOUS FUNCTION (NAMELESS FUNCTION)
    1. SYNTAX: var variable_name=function(){
      statments;
      }
      variable_name();
    2. FUNCTION WITH PARAMETERS var variable_name=function(para1,par2,....,paraN){ statments; } variable_name(arg1,arg2,...,argN);
    3. FUNCTION WITH RETURN TYPE var variable_name=function(para1,par2,....,paraN){ statments; return statments; } variable_name(arg1,arg2,...,argN);
  3. ARROW FUNCTION: ALTERNATIVE OF NAMED FUNCTION WITH SYMEATIC DIFFERENCE
    1. SYNTAX:var variable_name=() => (FAT ARROW){ statments; } variable_name();
      2. ARROW FUCTION WITH PARAMETERS SYNTAX:var variable_name=(para1,para2,.............,paraN)=>{ statments; } variable_name(arg1,arg2,.............,argN); IF WE HAVE MULTIPLE PARAMENTERS WE NEED TO USE PARANTHESIS COMPULSORY
  4. IIFE FUNCTION
      1. using anonomous function
      2. using arrow function
      3. IIFE function without parameters
        SYNTAX (function value)(callback value)/li>
      1. IIFE function with parameters
        2. SYNTAX (function value(para1,par2))(callback value(para1+par2))
  5. HIGHER ORDER FUNCTION & CALL BACK FUNCTION
      function with anonomous function parameters
      function callback(){
      statements;
      }
      function higherorder(){
      statements;
      }
      higherorder(callback);
      function with arrow function parameters
      function callback()=>{
      statements;
      }
      function higherorder()=>{
      statements;
      }
      higherorder(callback);
